This is the program for the BBU client.

1. Compile the program using cmake.
2. Compile the generated Makefile.
3. Run the program. The input parameters are:
	1. Edge server ip address
	2. Edge server port
	3. CPRI data rate
	4. The amount of time the program is supposed to log
	5. The time the program will start. Due to a software bug, this feature is not implemented.
